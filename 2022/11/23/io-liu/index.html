2、IO

2.1 流的概念

    水流，气流，车流，电流，人流，各种元素的流动
    文件流：文字的流动，只认识二进制，都是01，010101010101数据的流动   
 大小：
    字符流（16位二进制）和字节流（8位二进制）
 方向：相对于内存来说
    输入流（硬盘读入内存），输出流（内存到硬盘）
   字节输入流，字节输出流，字符输入流，字符输出流
io:input-output传统的输入输出流
bio:blocking-io：阻塞的输入输出流
nio:new-io,non-blocking-io:非阻塞输入输出流
aio:nio的升级版，现在很少用

2.2 字节流

字节输入流：
    is.read():返回的是一个字节的数据对应的整数值，如果读完了，返回的是-1
一个字节一个字节读，这种读法数据量很大的时候就很慢。

2.3 字符流

2.3.1 编码的发展

​ 计算机刚出来—AscII码—满足美国人自己使用 0-127

​ 计算机传到西欧—- Iso-8859-1 西欧文 128-255

​ 中国 GB2312 —只收录了常见的几千个字 很多字没有 —奇特现象 身份证有两个字是打印的， 有个字手写上去的

​ GBK—–GB18030

​ BIG5 — 大五码 香港台湾

​ ANSI：不是编码——这个是微软搞的一个标准，本地编码—-

国际标准化组织：

​ UTF编码：万国码，utf-8

​ UTF-32:任意字符都是使用32位二进制来表示。

​ UTF-16:任意字符都是使用16位二进制来表示。

​ UTF-8:可变长编码

例如：

   码值： 20013
   码值的十六进制:4e2d
   码值的二进制： 01001110，00101101   15位

3.3 聊天室

代码：

服务器代码
public class Server {
    List<SocketTask> clients = new ArrayList<>();

    public void start() {
        try {
            // 开启一个服务器
            ServerSocket server = new ServerSocket(8888);
            System.out.println("服务器启动啦。。。。。。");
            // 不断接收客户端的连接
            while (true) {
                // 监听客户端的连接
                Socket socket = server.accept();
                System.out.println(socket.getInetAddress().getHostAddress() + "连接了服务器");
                // 服务器就开辟一个新的线程来维护你这个客户端的聊天任务
                SocketTask st = new SocketTask(socket);
                // 现先放任务到集合还是先启动
                clients.add(st);
                new Thread(st).start();
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }

    class SocketTask implements Runnable {
        Socket socket;// 客户端是谁
        String name;// 客户端的名字是什么
        // to-do
        public void run() {
            try {
                // 编写任务--先接收一次名字，然后发送一个欢迎信息给所有目前聊天室的人
                InputStream is = socket.getInputStream();
                // 读写使用BufferedReader---BufferedWriter
                BufferedReader br = new BufferedReader(new InputStreamReader(is));
                String name = br.readLine() + "[" + socket.getInetAddress().getHostAddress() + "]";
                this.name = name;
                // 发送一个欢迎信息给所有的客户端
                sendMsg(name + "加入了聊天室");
                // 不断接收这个socket发过来的消息，并转发给所有人
                while(true) {
                    //读取这个客户端发的消息
                    String msg = br.readLine();		
                    sendMsg(name+":"+msg);//广播给所有的客户端
                    if(msg.equals("exit")) {//意味着客户端要退出聊天室
                        socket.close();
                        //从集合中把它移除
                        clients.remove(this);
                        break;//线程任务完成
                    }					
                }
            } catch (IOException e) {
                System.out.println(name+"退出了聊天室");
                clients.remove(this);
                sendMsg(name+"退出了聊天室");
            }
        }
        // 这个方法是把服务器的消息发送到所有客户端
        private void sendMsg(String msg) {
            try {
                for (SocketTask st : clients) {
                    BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(st.socket.getOutputStream()));
                    bw.write(msg);
                    bw.newLine();//要写一个换行
                    bw.flush();
                    //bw不能close--如果close就会关闭socket
                }
            } catch (IOException e) {				
                e.printStackTrace();
            }
        }
        public SocketTask(Socket socket) {
            this.socket = socket;
        }
    }

    // 入口转移
    public static void main(String[] args) {
        new Server().start();
    }
}
